#### Слияние таблиц
```pd.merge(left, right, on="", how="", left_on="", right_on="")```
```left.metge(right, on="", how="", left_on="", right_on=)```

Оба эти варианта выполнят слияние 2х таблиц
- left -- Левая таблица
- right -- Правая таблица
- on -- общий столбец у двух таблиц по которому происходит слияние
- left_on, right_on -- если у столбцов по которым нужно происзвести слияние разные имена в таблицах
- how -- способ слияния:
    - 'inner' -- Оставляет только те строки, ключи которых есть в обеих таблицах
    - 'left' -- Берет все строки из левой таблицы и добавляет данные из правой там, где есть совпадения. Если совпадений нет, ставится NaN
    - 'right' -- Работает зеркально левому: сохраняет все строки из правой таблицы.
    - 'outer' -- Объединяет все данные. Если ключ есть хотя бы в одной таблице, строка попадет в результат. Пропущенные данные заполняются NaN

#### Группировка
```tab.groupby("column")```

Группирует строки при одинаковом значении в заданом столбце **column**. Применяется например для объединения информации о предметах из одного заказа
- tab -- таблица
- column -- столбец

```.agg(new_col=(old_col, method), ...)```

Применяет функцию к колонке данных
- new_col -- имя новой колонки
- old_col -- имя колонки откуда берутся данные
- method -- операция применяемая к столбцу. (min, max, mean, median, std, var, sem, sum, count, size, nunique, first, last, quantile)

```.assign(new_col=val)```

Добавляет новый столбец или изменяет существующий
- new_col -- имя новой колонки
- val -- значения столбца(обычно через lambda)

```.transform(method)```

Делает преобразование внутри группы и применяет его ко всем элементам группы чтобы оставить исходный размер
- method -- операция применяемая к столбцу в группе. (sum, mean, max, min, count, first, last, rank)

#### Скоринг
```pd.cut(col, bins=4, labels=[1,2,3,4])```

Добавляет признак, разделяя значения на интервалы. Сортирует их по возрастанию просто разбивает растояние между максимальным и минимальным на интервалы
- col -- имя столбца по которому разбивается
- bins -- количество подгрупп(бинов)
- labels -- названия подгрупп

```pd.qcut(col, q=4, labels=[1,2,3,4])```

Добавляет признак, разделяя значения на группы по квантилям. Распределить данные так, чтобы в каждом интервале оказалось примерно одинаковое количество записей. Может плохо работать при слишком неравноменых данных
- col -- имя столбца по которому разбивается
- q -- количество подгрупп
- labels -- названия признаков

#### Сводная таблица
```.pivot_table(index=col, columns=col, values=col, fill_value=0)```
Создание сводной таблицы
- index -- что берется за строки
- сolumns -- что берется за столбцы
- values -- что берется за значения
- fill_value -- что ставить вместо пустых значений
